# Image Viewer - 图片查看器

## 项目概述
这是一个基于 PyQt6 和 OpenCV 实现的图片查看器，可用于加载、显示和交互式操作图像。支持平移、缩放以及双击获取图像像素 RGB 值，适合作为图像处理、计算机视觉学习和实验项目的基础工具。

---

## 功能特点
- **图片加载**：通过按钮选择 JPG/PNG/JPEG 格式图片进行显示。
- **图像显示**：使用 QLabel 显示图像，保持居中对齐。
- **鼠标平移**：按住鼠标拖动图片实现平移。
- **鼠标滚轮缩放**：以鼠标当前位置为中心进行放大或缩小。
- **双击输出 RGB**：双击图片任意位置，控制台输出该点像素的 RGB 值。
- **实时刷新**：拖动和平移操作会实时更新显示。

---

## 依赖环境
- Python 3.9+
- PyQt6
- OpenCV (cv2)
- NumPy
---
**安装依赖示例**：
pip install PyQt6 opencv-python numpy

## 运行：
python image_viewer_template.py

## 实现关键：
- **鼠标平移**：计算鼠标移动偏移量 (dx, dy)，构造平移矩阵，通过矩阵左乘累积变换，实现图像的连续平移。
- **鼠标滚轮缩放**：根据滚轮滚动角度计算缩放系数 scale，以鼠标位置为缩放中心，构造变换矩阵，先平移到原点 → 缩放 → 再平移回中心，实现以鼠标为中心的缩放。
- **双击输出 RGB**：通过逆矩阵将窗口坐标映射到原图坐标，获取精确像素值。
